- Submit private personal data about living persons (phone, address,

- Scrape the site or the API at a rate that materially degrades service

## Accuracy disclaimer

wikipath is an evolving open-data project. Records derived from
Wikipedia, Wikidata, and LLM extraction are best-effort and may contain
errors. Every fact has a `source_sentence` quote, a confidence score,
and a source-badge — use those to judge a claim before acting on it.
We do not warrant the accuracy, completeness, or fitness for any purpose
of any record.

If you find an error, please report it (see TAKEDOWN.md or use the
"Báo lỗi" button on the affected record).
